## *Eclipse-on-Mac* keymap for IntelliJ IDEA

Emulates the *Eclipse IDE* keymap on OS X (Mac OS) by replacing all instances of
`Ctrl` with `âŒ˜`.

It's originally based on the keymap "Eclipse" of *IntelliJ IDEA* shipped with:

[http://git.jetbrains.org/?p=idea/community.git;a=blob_plain;f=platform/platform-resources/src/idea/Keymap_Eclipse.xml;hb=HEAD](http://git.jetbrains.org/?p=idea/community.git;a=blob_plain;f=platform/platform-resources/src/idea/Keymap_Eclipse.xml;hb=HEAD)

To install this keymap clone this repository with `git clone`, after then type
`./install.sh`. Alternatively, you can copy the file `Eclipse on Mac.xml` manually into your
IDEA preferences folder `~/Library/Preferences/IdeaIC12/keymaps` (depending on
your IntelliJ IDEA version).
